# /CODE_OF_CONDUCT.md
# 说明：基于简化版 Contributor Covenant 的中文行为准则，适用于本项目的所有参与者。

## 基本原则
- 尊重彼此：在讨论中保持礼貌，不进行人身攻击或歧视性言论。
- 包容多样性：欢迎不同背景、经验与观点的贡献者参与协作。
- 透明沟通：反馈问题或提出建议时，请提供足够上下文，便于他人理解。

## 我们的承诺
- 维护开放友善的社区环境，让每个人都能安全地参与贡献。
- 积极响应合理的反馈，如遇冲突将尽力协调并给出解决方案。
- 及时处理违反准则的行为，包括警告、临时限制或移除贡献权限。

## 适用范围
- 本准则适用于项目仓库、Issue、Pull Request、文档与任何线上线下活动。
- 当代表项目社区参与其他平台或公共活动时，也应遵守相同标准。

## 反馈渠道
- 如需举报违规行为，请通过 Issue 或私信维护者，并提供相关证据。
- 维护者收到举报后会在合理时间内回复，并确保举报人隐私不被泄露。

## 执行流程
1. 维护者首先评估行为影响，必要时收集更多信息。
2. 如行为轻微，将给出提醒并记录；如行为严重，将采取相应措施。
3. 若当事人对处理结果有异议，可提交进一步说明，维护者将复核。

## 致谢
- 本准则参考了 [Contributor Covenant](https://www.contributor-covenant.org/) 的核心理念，并结合本项目特点简化。
