# 配置参数指南（占位）

本文件将用于详细说明 Stage2048 的所有可配置参数，包括运行时调优项与跨平台共享的默认值。当前为占位内容，后续轮次会补充完整说明与示例代码。

## 参数说明（预览）
- `boardSize`：当前关卡对应的棋盘尺寸，将由关卡管理器在运行时写入。
- `gap`：Canvas 绘制时每个方块之间的像素间距，可在 Web 与小程序中共享。
- `colors`：映射数字到前景色与背景色的配置，后续会提供默认色板与自定义建议。
- `randomTileWeights`：新生成数字的概率分布，基础默认值为 `{1:0.9, 2:0.1}`，以支持从数字 1 起步的规则。
- `levelStartSize`：关卡起始的棋盘尺寸，默认从 `2`（即 2×2）开始。
- `carryScore`：控制通关时是否累计总分的布尔开关，默认开启。
- `targetFn(size)`：根据棋盘尺寸计算目标值的函数，默认实现为 `size => 2 ** (size + 3)`。
- `randomTileWeightsBySize`：针对不同棋盘尺寸覆盖 `randomTileWeights` 的配置表。

## 目标函数示例
```js
// 函数签名示例：传入棋盘尺寸，返回该关卡的目标数值。
function targetFn(size) {
  return 2 ** (size + 3);
}
```
