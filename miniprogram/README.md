# 微信小程序版本说明（占位）

本目录将承载微信小程序版本的实现。当前阶段仅提供基础占位文件，方便在微信开发者工具中导入并预览。

## 导入步骤
1. 打开微信开发者工具，选择“导入项目”。
2. 目录选择本仓库的 `miniprogram/` 文件夹。
3. 将 `project.config.json` 中的 `appid` 替换为你自己的小程序 ID。
4. 点击“导入”后即可看到占位页面，其中包含 Canvas 容器与后续开发提示。

## 未来扩展
第 4 轮将接入核心逻辑与关卡管理，实现 Canvas 绘制、手势交互以及关卡推进。后续还会在 `miniprogram/core/`（计划中新建目录）中复用 `core/` 的通用逻辑。
